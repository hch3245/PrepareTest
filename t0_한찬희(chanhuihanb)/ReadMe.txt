편의성 개선
1. 공을 생성할 때 겹치는 문제 방지
	- 새로 생성되는 공이 기존 공들과 겹치는지 체크하고 방지합니다.
	- 더 이상 공을 생성할 수 없다고 판단될 경우, 최대치까지만 공을 생성하고 에러 창을 띄웁니다.
	- 에러 창의 경우, ImGui 메인 창을 클릭하면 자동으로 닫아지도록 하여 편의성을 높였습니다.

추가 기능
1. 탄성 계수 적용
	- 탄성 계수를 조절하여 완전 탄성(반사)에서 비탄성(에너지 손실) 충돌까지 다양한 효과를 적용할 수 있습니다.
	- ImGui에서 탄성 계수를 직접 입력하여 실시간으로 변경할 수 있습니다.

2. 자력 적용
	- 각 공에 자기 극성을 부여하여 공들이 서로 밀거나 당기는 힘을 적용합니다.
	-  ImGui에서 해당 기능의 활성화 여부와 세기를 설정할 수 있습니다.

3. 마우스 클릭으로 특정 공 잡아서 움직일 수 있는 기능
	- 마우스로 특정 공을 클릭하면 해당 공을 선택하여 움직일 수 있습니다.
	- 드래그하는 동안 공이 마우스를 따라 이동하며, 마우스 버튼을 놓으면 마지막 이동 속도가 공의 속도로 적용됩니다.
	- ImGui 창이 마우스 입력을 가리는 상황을 고려하여 기능을 구현했습니다.

4. 마우스 클릭으로 특정 위치에 공 생성 및 제거 기능
	- 사용자가 마우스로 특정 위치를 클릭하여 공을 생성할 수 있습니다.
	- 공 생성 시 기존 공들과의 겹침 여부를 확인하며, 겹칠 경우 생성이 방지됩니다.
	- 제거 모드에서는 공이 하나만 남아있을 때 삭제가 불가능하도록 처리하여 안전성을 확보했습니다.
	- 별도의 ImGui를 통해 추가 및 제거 모드를 전환할 수 있으며, 오류 메시지도 함께 표시됩니다.
